features

Everything you'd want.

Built for people who want maximum control over how their games launch, without the overhead of a full platform.

SteamDB version checking
Automatically checks your installed build ID against the latest Steam build on startup.
Custom launch configs
Create named quick-launch profiles per game for different parameters. One click to switch.
Category management
Organize your library into categories. Drag and drop games between them. Collapsible sections keep things clean.
Steam protocol launch
Launch via steam://rungameid/ for full workshop mod support, or run the executable directly for DRM-free games.
Auto-updater
Checks for new versions of CLauncher on launch and downloads updates silently in the background. Restart to apply.
Minimal UI
Dark, frameless window with custom titlebar. Searchable sidebar, inline param editing, keyboard-friendly throughout.
